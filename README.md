# Gody Corominas - Dossier Artístico  Este proyecto es un dossier artístico digital y profesional para **Gody Corominas**, músico, autor y productor argentino con más de 40 años de trayectoria. La aplicación funciona como un portafolio interactivo de alto impacto visual, diseñado para presentar su carrera a nivel nacional e internacional.
